## Data Description

The dataset used in this project is organized into three main parts based on their functionality:

### Part 1: Main Instance Files
The instance files are named using the format `Ins_i_j_k(T)_m101`.
-   **i**: Number of patients.
-   **j**: Number of caregivers.
-   **k**: Number of time slots.
-   **T**: Number of provided time slots.
-   **m**: Type of patient, which can take three values: `C`, `R`, or `RC`.

### Part 2: Parameter Configuration
The folder (or section) named `parameter` contains the parameter data used during the experiments.

### Part 3: Sensitivity Analysis Data
The folder (or section) named `sensitivity_analysis` contains the data pertaining to patients, caregivers, and time slots, which is specifically prepared for conducting sensitivity analysis.
